# UniPort Student Risk System - Frontend

## Overview

Next.js frontend for the UniPort Student Risk System. It includes login, registration, landing page, and an admin-style dashboard.

## Setup

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```

## Environment

No additional frontend environment variables are required for local development.

## Build

```bash
npm run build
```
